# Vim

#### Navigation 

###### Move to top of file
```
gg
```

###### Move to end of file
```
G
```

###### Beginning of next word
```
w
```

###### End of next word
```
e
```

###### Beginning of line
```
^
```

###### End of line
```
$
```

#### Editing

###### Insert mode
```
i
```

###### Save and quit
```
:wq
```

###### Repeat last action
```
.
```

###### Delete everything until the end of this line
```
D
```
